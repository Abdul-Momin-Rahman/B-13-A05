1️⃣ What is the difference between var, let, and const?

Answer () =>

var  is function scoped , we can redeclare it and it is hoisted but has some issues like using before initialization  , so we don’t use it often.

Let and const are more modern ES6 Properties, where both are block scoped and can not be redeclared after initialization and let can be reassigned while const can not.

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer ()=>

map(),filter(),forEach() all are modern ES6 array methods of JavaScript ,

map() returns a new array after applying a specific function to each element in the original array.

filter() returns a new array that includes only those elements that passed the checking of a specific condition for each element in the original array.

forEach() is a method which goes through every element in an array and applies a function to them without returning anything..
